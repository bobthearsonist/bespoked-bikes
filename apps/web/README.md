# Frontend

Simple static HTML frontend for the BespokedBikes API.

## Quick Start

```bash
# With Docker (Recommended)
docker-compose up frontend

# Access at http://localhost:3001
```

## What It Does

- Displays "BespokedBikes Frontend" header
- Links to the API documentation at `/scalar/v1`
- Served via nginx

## Files

- `public/index.html` - Main HTML page
- `nginx.conf` - Nginx configuration
- `Dockerfile` - Container definition
